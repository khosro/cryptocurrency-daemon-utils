1.First set the following based on your settings :

BasePath = ""

Specify BtcPay_Src and BasePath. BasePath is the path that all cryptocurrency softwares and cofings are located.
And also all software located in its home directory with bin folder that contains daemon .For example bitcoin core located in %BitCoin_Home% and in %BitCoin_Home% we have bin folder that contains bitcoin core daemon(bitcoind.exe).The same applied for all other softwares.

Then Extract the content of cryptocurrency-daemon-utils_daemon.zip to root of project.


setx   Btcpay_Data_Home     "BasePath\btcpayserver"
setx   BtcPay_Src           "Root folder of your BTCPay Server source code"

setx   BitCoin_Data_Home    "BasePath\bitcoin"
setx   BitCoin_Home         "BasePath\bitcoin\bitcoind"

setx   Ethereum_Home        "BasePath\Ethereum" 

setx   LiteCoin_Data_Home   "BasePath\litecoin"
setx   LiteCoin_Home        "BasePath\litecoin\litecoind"


You can use the followning bat files : 

 %LiteCoin_Home%\litecoinStart.bat
  
 %BitCoin_Home%\bitcoinStart.bat
 %BitCoin_Home%\bitcoinclicmd\bitcoinclicmd.bat
 %BitCoin_Home%\btc-explorer.bat
 
 %Btcpay_Data_Home%\NBXplorer-run.bat
 %Btcpay_Data_Home%\btcpayserver-run.bat
  
 %Ethereum_Home%\devnet\bootnodeStart.bat
 %Ethereum_Home%\devnet\node1.bat
 %Ethereum_Home%\devnet\node2.bat


2.Update : In new version of bat files we do not the following commands(section 2), because we use the above environment variables in bat files.

set "Path=%BitCoin_Home%\bin;%Path%"
set "Path=%Ethereum_Home%\bin;%Path%"
set "Path=%LiteCoin_Home%;%Path%"
setx Path "%Path%"

Or set them in PATH system variable
